1.

editable llm transcript lab

An environment for editing any visible part of an LLM conversation: system, user, assistant output, and an explicit reasoning scaffold. The goal is to create SOTA intended chats and eval targets by making the transcript itself the source of truth, then continuing from that edited state.

It could become useful for prompt evals, agentic trajectories, target rationales, and understanding what specific context produces which behavior.
